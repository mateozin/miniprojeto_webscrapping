# miniprojeto_webscrapping
Um algoritimo que por enquanto apenas encontra noticias com mais de 30 caracteres do g1.globo. O usuário pode utilizar o Enter para ir para a próxima notícia ou utilizar o q + Enter para parar. As notícias ficam salvas sem repetições no arquivo noticias_g1.csv
# Como Usar?
Crie um ambiente virtual na pasta do arquivo e ative-o
```
python -m venv .venv
.venv\Scripts\activate
```
Instale os "requirements.txt"
```
pip install -r requirements.txt
```
Execute o algoritimo estando na pasta correta no terminal utilizando o comando 
```
python scrapping.py
```
Utilize a tecla Enter no terminal para ir para a próxima notícia ou utilize o Q + Enter para cancelar.
