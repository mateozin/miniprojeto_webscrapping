# miniprojeto_webscrapping
Um algoritimo que por enquanto apenas encontra noticias, com mais de 40 caracteres, do g1.globo e cria um arquivo listando apenas as 10 primeiras encontradas.
# Como Usar?
Crie um ambiente virtual na pasta do arquivo e ative-o
```
python -m venv .venv
.venv\Scripts\activate.bat
```
Instale os "requirements.txt"
```
pip install -r requirements.txt
```
Execute o algoritimo estando na pasta correta no terminal utilizando o comando 
```
python scrapping.py
```
